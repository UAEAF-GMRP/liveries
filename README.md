# UAEAF GMRP Liveries

This repository contains official **United Arab Emirates Air Force (UAEAF)** liveries used in **GMRP (GeoFS Military Roleplay)**.

The UAEAF GMRP group conducts structured aviation roleplay within **GeoFS**, focusing on realism, coordination, and authentic air force representation.

All liveries included here are intended for use in GMRP sessions to maintain visual consistency and realism across aircraft.

Learn more about the real-world United Arab Emirates Air Force:
https://geofs.fandom.com/wiki/United_Arab_Emirates_Air_Force

## Usage

### Steps to Upload a New Livery

#### 1. Upload the image
Upload the livery image to **[this GitHub repository](https://github.com/UAEAF-GMRP/liveries)** or **[IMGBB](https://imgbb.com/)**

#### 2. Copy the direct image link
Open the uploaded image in a new tab and copy the **direct image URL**

**Important**
- The link **must end with the file format**
  - `.jpg`
  - `.png`
  - `.webp`

**Example**
```
[https://jollycontrarian.com/images/6/6c/Rickroll.jpg](https://jollycontrarian.com/images/6/6c/Rickroll.jpg)
```

#### 3. Download the current JSON file
Open the **[current JSON file](https://raw.githubusercontent.com/UAEAF-GMRP/liveries/refs/heads/main/uaeaf.json)** and save it to your computer  
(**Ctrl+S** or equivalent on Mac)

#### 4. Validate the JSON file
Open the **[livery validator](https://uaeaf-gmrp.github.io/liveries/livery-validate.html)** and upload the downloaded JSON file

#### 5. Add the new livery
Scroll to the **Add Livery** section and click **Add Livery**

Fill the following fields:
- Select aircraft
- Paste livery image link
- Download updated JSON

#### 6. Upload the updated JSON to GitHub
Replace the existing JSON file in the repository with the updated version
